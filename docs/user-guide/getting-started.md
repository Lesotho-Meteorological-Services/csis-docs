# Getting Started

## Overview

The CSIS supports collaborative preparation, review, approval, and dissemination of climate services products.

Typical user roles may include:

- Meteorologist
- Sector contributor
- Reviewer
- Approver
- Dissemination officer
- System administrator

## Basic user journey

1. Log in to the platform
2. Access the relevant workspace or module
3. Create or open a product
4. Contribute content
5. Submit for review
6. Publish or disseminate once approved

## Good practices

- Use the correct template for each product type
- Follow naming conventions
- Save work regularly
- Check approval status before dissemination
